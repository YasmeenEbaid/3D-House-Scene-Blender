# 🏠 3D House Scene - Blender Project
This project is a 3D house scene created using Blender as part of a Computer Graphics assignment. It demonstrates the use of core 3D design concepts including modeling, texturing, lighting, and animation.
---
## 🚀 Features
- 🧱 3D Modeling of a house and environment  
- 🎨 Materials and textures application  
- 💡 Lighting setup for realistic rendering  
- 🎥 Camera positioning and movement  
- 🎬 Basic animation (if included)  
---
## 🛠️ Tools Used
- Blender  
---
## 🎯 Learning Outcomes
- Understanding of 3D modeling workflow  
- Applying lighting and camera techniques  
- Creating realistic scene rendering  
- Basics of animation in Blender
